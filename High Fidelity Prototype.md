# Anahí Cacau E-Commerce: High-Fidelity Prototype Documentation

This document outlines the high-fidelity prototype for the Anahí Cacau e-commerce website. It details the visual design system and the core user interface screens that compose the user journey.

## 1. Style Guide

The visual identity of the Anahí Cacau website is built on a foundation of warm, earthy tones and clean, modern typography. This combination aims to evoke a sense of natural quality, tradition, and premium craftsmanship.

<img width="407" height="696" alt="Style_Guide" src="https://github.com/user-attachments/assets/aa754740-51f5-4391-8919-c6f2bf30637f" />

### 1.1. Typography

The primary typeface used is **Work Sans**, chosen for its clarity and versatility across different weights. This ensures excellent readability for all text, from bold headlines to detailed paragraphs.

| Usage | Font | Weight | Size |
| :--- | :--- | :--- | :--- |
| Hero Titles | Work Sans | Extra Bold | 84px |
| Section Titles | Work Sans | Bold | 44px |
| Body & Paragraphs | Work Sans | Regular | 22px |

### 1.2. Color Palette

The color scheme is inspired by the natural elements of cacao and the Brazilian landscape. A deep red and burnt orange serve as primary action colors, while a palette of rich browns and light beige creates a warm and inviting background.

| Color | Hex Code |
| :--- | :--- |
| Intense Red | `#C62828` |
| Burnt Orange | `#E65100` |
| Light Beige | `#F5E6D3` |
| Dark Brown | `#3E2723` |
| Medium Brown | `#6D4C41` |

## 2. Prototype Screens

The following sections present the key screens of the high-fidelity prototype, showcasing the user flow from browsing to checkout.

### 2.1. Home Page

The home page serves as the main entry point, immediately immersing the user in the brand's story. It features a compelling hero section, highlights of popular products, a brief introduction to the company's mission, and a glimpse into the natural ingredients used.

<img width="327" height="871" alt="Screenshot 2025-11-23 132653" src="https://github.com/user-attachments/assets/b476585e-c2db-44ee-8bd3-2ff3b0e3fead" />

### 2.2. Products Page

This screen displays the full range of products available for purchase. Users can filter and sort items, with clear pricing and visuals. Product cards include key information and an option to quickly add items to the cart. Badges for "New," "Sold Out," or "Sale" provide immediate status updates.

<img width="1051" height="823" alt="Screenshot 2025-11-26 055346" src="https://github.com/user-attachments/assets/6f9e1462-12a7-488f-9c02-78b0f08e9e19" />


### 2.3. Product Detail Page

When a user selects a product, they are taken to this detailed view. The page features a larger product image, a descriptive text, price, and a prominent "Add to Cart" button. Nutritional information is also available, providing transparency to the customer.

<img width="719" height="784" alt="Screenshot 2025-11-29 105701" src="https://github.com/user-attachments/assets/757b6ee5-fdef-4e9e-8e6e-6da9e0f8a42f" />

### 2.4. My Cart

The shopping cart is implemented as a slide-out sidebar for easy access without leaving the current page. It provides a summary of the items added, allows for quantity adjustments, and shows the subtotal. A progress bar incentivizes users to reach a free shipping threshold.

<img width="678" height="747" alt="Screenshot 2025-11-29 105726" src="https://github.com/user-attachments/assets/3f830799-4bee-43cf-b041-275624016f27" />

### 2.5. Order Summary Page

This page provides a full view of the shopping cart before checkout. It displays all items, unit values, and totals. Users can see suggested products and have a clear call-to-action to proceed to checkout or continue shopping. It also includes options for delivery or pickup.

<img width="686" height="746" alt="Screenshot 2025-11-29 105750" src="https://github.com/user-attachments/assets/a3b5f503-1bce-4d5e-bb37-c63c0f9611e6" />

### 2.6. Login & Register Pages

To complete a purchase or manage an account, users can log in or create a new account. The login and registration forms are designed for simplicity and ease of use, featuring clean layouts and clear input fields, consistent with the overall site aesthetic.
